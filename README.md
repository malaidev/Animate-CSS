# BeautyBooking

A beauty bookings web application for hair, nails, spa and etc. appointments.  :calendar: :nail_care:

- ASP.NET Identity System
- MVC Areas with Multiple Layouts
- Razor Pages, Sections, Partial Views
- View Components
- Repository Pattern
